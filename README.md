# AWS EC2 Web Server Project

## Overview
This project demonstrates deploying a web server using AWS EC2.

## What I did
- Launched EC2 instance (Ubuntu)
- Connected using EC2 Instance Connect
- Installed Apache web server
- Configured security groups (HTTP & SSH)
- Hosted a live website

## Technologies
- AWS EC2
- Ubuntu Linux
- Apache

## Result
Successfully deployed a live website accessible via public IP.
